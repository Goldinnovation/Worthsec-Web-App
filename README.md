# Learning Project: Worthsec Web App

Worthsec is a Web application that offers users the opportunity to create, share, discover, exchange, and participate in exciting events.





### Key Features:
- CRUD
- Explore
- Share 
- Chat
- Invite
- Join 
 

### Tech Stack:
- Node.js 
- Next.js - Frontend 
- Express.js - Backend 
- Database - Postgresql
- ORM - Prisma
